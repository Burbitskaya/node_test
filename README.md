# Cat Facts 
Скрипт для поиска самого короткого факта о котах с использованием API Cat Facts Ninja.

## Установка и запуск

Клонируйте репозиторий:
```
git clone https://github.com/Burbitskaya/node_test.git
cd test_node
```
Установите зависимости:
```
npm install
```

Запустите скрипт:
```
npm start
```
